# 專案協作入口

## 讀取順序

1. `CURRENT_STATE.md`
2. `USER_REQUIREMENTS.md`
3. `GOTCHAS.md`
4. 依任務讀取 `ARCHITECTURE.md`、`VERIFY.md`、`COMMANDS.md`

## 專案重點

- 這是大葉大學校園花卉互動導覽的純前端網站。
- `daylily.html` 是金針花介紹與沉浸式探索頁。
- 免費 3D 地圖使用 MapLibre GL JS、OpenFreeMap 與 OpenStreetMap 資料。
- `images/daylily-bloom.glb` 是網站使用的完整金針花植株模型。
- 金針花地圖標記是展示位置，不能當成精準導航資料。
